# tic-tac-toe

Sample Tic Tac Toe app using DragonRuby and Sinatra
Demonstrates local multiplayer, vs Computer, and online

## Running

This is in DragonRuby v4. Place your dragon ruby files appropriately and start as normal

To run the API

```
cd api
bundle install
bundle exec puma
```
